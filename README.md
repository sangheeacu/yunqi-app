# Yun Qi 運氣 Clinical Analyzer

**Wu Yun Liu Qi 五運六氣 — Full Clinical Analysis Tool**

By Sang Hee Park, PhD — Blossom Skin & Health, Sydney

---

## Features

- 🔢 **Any year 1924–2103** — automatic analysis
- 🌏 **Northern & Southern Hemisphere** date switching
- 📊 **10 Yun-Qi segments** — 初運初氣 through 終運終氣
- 🌤 **Weather forecast** per segment
- 🏥 **Disease prediction** per segment
- 📍 **Acupuncture points** per segment
- 🌿 **Herbs & Formulae** per segment
- 📄 **PDF export** — print or save
- 📱 **PWA** — add to home screen, works offline

---

## Deploy on Vercel

1. Fork or upload this repository to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click **New Project** → Import from GitHub
4. Select this repo → **Deploy**
5. Your app is live at `https://your-app.vercel.app` ✅
